## Описание проекта
Этот проект родился из разбора старых зарисовок времен моего студенчества, когда в рамках научной работы я реализовывал алгоритм старой китайской игры Вэйци она же Го с которой познакомил меня мой научный руководитель.
Го — это стратегическая настольная игра, в которой два игрока по очереди размещают камни на доске, пытаясь окружить территорию и захватить камни противника. Данный проект реализует базовую версию игры с графическим интерфейсом.

---

## Особенности

- **Графический интерфейс**: Используется библиотека `pygame` для отрисовки доски и камней.
- **Правила игры**:
  - Размещение камней.
  - Захват групп камней.
  - Правило ко (запрет повторения позиции).
- **Подсчет очков**: Учитываются захваченные камни и коми.
- **Меню**: Возможность выбора размера доски (9x9, 13x13, 19x19).
- **Управление**:
  - Клик мыши для размещения камня.
  - Клавиша `P` для пропуска хода.
  - Клавиша `U` для отмены хода.
  - Кнопка "Выход" для завершения игры.

---

## Требования

Для запуска проекта необходимо установить следующие зависимости:

- Python 3.7 или выше.
- Библиотека `pygame` для графического интерфейса.
- Библиотека `numpy` для работы с массивами.
- Библиотека `networkx` для поиска групп камней.
